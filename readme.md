# 🇮🇳 FULL OTA INDIAN FIRMWARE – OnePlus Nord 4 (CPH2661)

This repository contains Unofficial full OTA firmware packages maintained by LuffyOP for the OnePlus Nord 4 (Indian variant – CPH2661).

These files can be used for manual flashing, recovery, or firmware restoration purposes.

---

## 📦 What’s Inside?

- Official OnePlus full OTA firmware  
- Split archive parts (.0001, .0002, etc.)  
- Flashable full OTA package after extraction  

---

## 🛠 How to Use

1. Download all parts of the firmware and place them in the same folder  
2. Install 7-Zip on your PC  
3. Right-click on the file ending with `.0001` → Extract here  
   - If prompted to select the next parts (.0002, .0003…), choose them  
4. After extraction, you will get one large file with a random name  
   → This is your full OTA firmware package  
5. Flash it using a Fastboot Firmware Flasher or your preferred flashing method  

---

## ⚠️ General Things to Remember

- ✅ Cross-region flashing is supported  
  You can flash IND firmware on Global devices and even re-lock the bootloader if everything is stock.

- ✅ Boot & init_boot images are the same for all regions  
  Just make sure the image version matches the ROM version you are currently running.
